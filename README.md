# WeedSeeds — демо сайт (неоморфный чёрно‑зелёный)

## Что внутри
- Адаптивный хедер:
  - ПК: логотип + пункты меню + поиск (без «бургер»-кнопки)
  - Мобилка: слева бургер, по центру/левее лого, справа поиск; пункты — в выезжающем меню
- Поле баннеров (карусель):
  - Поддержка фото и видео (mp4/webm)
  - Автоперелистывание + точки + стрелки + свайп на телефоне
  - Ограничение высоты: до 25vh на мобиле (≈ 1/4 экрана) и до 50vh на ПК (≈ 2/4)
  - Баннеры кликабельны (href)
  - Добавление/удаление баннеров через демо админку (#/admin)
- Категории (квадратные блоки, закругления, «логотип» внутри)
- Популярные сорта (3 примера)
- Карточка товара:
  - Галерея фото формата 4:3 + миниатюры + свайп
  - Характеристики, описание
  - Заказ в Telegram (открывает t.me/<username>?text=...)
- Отзывы с модерацией:
  - Пользователь отправляет → статус pending
  - В админке можно одобрить → появляется на странице товара

## Запуск
Это статический сайт. Открой `index.html` в браузере, но лучше через локальный сервер:
- VS Code → Live Server
- или `python -m http.server` в папке проекта

## Как вставить свой логотип и картинки
- Замени файлы в `/assets`:
  - `logo.png`
  - `banner-1.jpg`, `banner-3.jpg`, `banner-2.mp4` (или webm)
  - `p1-*.jpg` и т.д.
- Или добавь баннеры через `#/admin` (PIN по умолчанию 1111)
